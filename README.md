# 长沙理工大学新生入学指南

一个面向长沙理工大学本科与研究生新生的手机优先静态网页指南。项目不依赖构建工具，适合直接部署到 GitHub Pages、学校社群公告、公众号菜单或 QQ/微信群文件入口。

## 功能

- 全文搜索：按问题、关键词、校区、事务类型快速查找。
- 身份筛选：支持本科、硕士、博士、非全日制。
- 校区筛选：支持云塘、金盆岭和通用信息。
- 分区导航：覆盖报到前、到校当天、宿舍生活、学习事务、校园办事、吃住行、社团成长、安全健康、常见坑点。
- 本地收藏：常用问题收藏保存在浏览器本地。
- 报到清单：高频任务可勾选，并支持打印为 PDF。
- 官方来源：流程类信息优先标注学校官方页面，未确认内容标记为待更新。

## 本地使用

直接用浏览器打开 `index.html`。不需要安装依赖，也不需要启动服务。

## 项目结构

```text
.
├── index.html    # 页面结构和常用官方入口
├── styles.css    # 响应式样式和打印样式
├── app.js        # 内容数据、搜索筛选、收藏和清单逻辑
├── README.md     # 项目说明
├── LICENSE       # 开源许可证
├── .nojekyll     # GitHub Pages 静态站点配置
└── .gitignore    # 排除本地噪音、构建产物和导出文件
```

## 维护方式

- 问题卡片在 `app.js` 的 `cards` 数组中维护。
- 报到清单在 `app.js` 的 `checklist` 数组中维护。
- 涉及时间、费用、报到地点、电话、材料要求的内容必须补充官方来源。
- 2026 级正式入学手册发布后，优先更新标记为“待 2026 入学手册更新”的卡片。
- 不采信未经核验的社群传言；经验建议应和官方流程分开表达。

## 部署到 GitHub Pages

1. 将本目录提交并推送到 `ceckkk7475/csust_guide` 的默认分支。
2. 在仓库 Settings → Pages 中选择从默认分支根目录部署。
3. 部署完成后，把 Pages 链接放到班级群、迎新群或公众号菜单。
